## CLI 命令格式（重要）
为了支持“断线后仍保持会话”，除 LOGIN 外的命令都需要携带 sessionToken。

- LOGIN <user> <pass>
- LOGOUT <token>
- READ <token> <path>
- WRITE <token> <path> <content...>
- MKDIR <token> <path>
- BACKUP <token> <path>
- SUBMIT_REVIEW <token> <operation> <path>

内置测试账号（可在 src/auth/Authenticator.cpp 修改）：
- admin/admin123
- editor/editor123
- reviewer/reviewer123
- author/author123

## 本地测试（Windows）
1) 构建：
- cmake -S server -B server/build
- cmake --build server/build --config Release --target server

2) 启动服务端：运行 server/build/Release/server.exe（默认端口 8080）

3) 启动客户端：python server/test/test_client.py
- 先输入 LOGIN 获取 token
- 后续直接输入 WRITE/READ/BACKUP/SUBMIT_REVIEW 等命令即可（客户端会自动注入 token）
